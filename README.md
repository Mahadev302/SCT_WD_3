#  Quiz Game Application

An interactive and visually engaging quiz game built using HTML, CSS, and JavaScript that allows users to answer multiple-choice questions and view their score at the end.

##  Features
- Dynamic multiple-choice questions
- Instant answer feedback (correct/wrong)
- Final score display
- Modern UI with animations
- Responsive design

##  Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
